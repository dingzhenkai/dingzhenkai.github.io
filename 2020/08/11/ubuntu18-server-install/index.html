记录一下我在服务器上装ubuntu18的步骤，以供下次参考。

准备u盘系统盘

  1. 下载ubuntu18 iso文件
    去官网 下载server版的ubuntu，注意这里的amd64就是x86-64
  2. Windows下载refus, 将iso刷入u盘

这样我们就得到了一个u盘系统盘。

安装

  1. bios选择u盘启动
  2. 选择安装ubuntu
  3. 选择语言 英语
